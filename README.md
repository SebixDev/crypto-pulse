# ⚡ Crypto Pulse

> Ein elegantes Echtzeit-Dashboard für die weltweit größten Kryptowährungen. Alle wichtigen Markttrends übersichtlich und im modernen Dark-Design auf einen Blick.

Crypto Pulse ist das Krypto-Schwesterprojekt zu **Market Pulse**. Es bietet eine hochperformante, visuell ansprechende Oberfläche, um Live-Preise, historische Trends und Marktkapitalisierungen der wichtigsten digitalen Assets ohne Ablenkungen zu verfolgen.

---

## ✨ Features

* 📊 **Echtzeit-Daten:** Live-Kursaktualisierungen für die wichtigsten Kryptowährungen.
* 📈 **Interaktive Charts:** Cleane Visualisierung von historischen Kursverläufen (Linien- und Candlestick-Charts).
* 🌗 **Modernes Dark-Design:** Ein augenschonendes UI-Konzept in tiefem Anthrazit mit dem markanten Crypto-Orange als Akzent.
* 🔍 **Blitzschnelle Suche:** Sofortiges Filtern nach Coins und Tickersymbolen (z. B. BTC, ETH).
* 📱 **Fully Responsive:** Optimiert für Desktop, Tablets und Smartphones.

---

## 🛠️ Tech Stack

* **Frontend:** HTML5, CSS3 (Modernes Grid & Flexbox Layout), JavaScript (ES6+)
* **APIs:** CoinGecko API & CoinMarketCap API
* **Charts:** Chart.js / TradingView Lightweight Charts (geplant)
